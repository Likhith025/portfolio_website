Tirumala Construction ERP (Freelance)

Nov 2024 – June 2025

Developed a custom ERP system for Tirumala Constructions as a freelance full-stack developer, to streamline operations including inventory management, vendor orders, expenses and income tracking, and automated financial reports like P&L and balance sheets. The system allows the client to monitor business operations in real-time, reduce manual errors, and enhance decision-making efficiency.

  • Designed backend services using Express.js and Mongoose for efficient CRUD operations on projects, inventory, and transactions.
  • Implemented JWT-based role access control with middleware for Admin, Manager, and Staff users.
  • Built a responsive React.js frontend for visualizing data, generating reports, and performing daily operations.
  • Automated P&L and balance sheet calculations based on live financial data, improving accuracy and decision-making.
  • Integrated GitHub CI/CD pipelines for automated deployment of frontend (Vercel) and backend (Render) with environment-specific configs.
  • Optimized database schema and queries to support multi-user concurrent operations seamlessly.
