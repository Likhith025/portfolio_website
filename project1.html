Defect Product Detection

Oct 2024 – Nov 2024

Developed a computer vision-based defect detection system to automatically identify faulty jar lids with high accuracy. The project combines YOLOv5 for object detection, OpenCV for image preprocessing, and Python for model training and evaluation. It aims to improve quality control in manufacturing by reducing manual inspection time and increasing defect detection reliability.

  • Implemented YOLOv5 for object detection, achieving precise identification of defective products.
  • Applied image preprocessing techniques using OpenCV such as noise reduction, contour detection, and adaptive thresholding to enhance detection quality.
  • Collected, annotated, and preprocessed a large dataset of jar lid images to train and validate the model.
  • Utilized Google Colab for GPU-accelerated training and experimentation, optimizing hyperparameters for maximum performance.
  • Designed modular Python scripts for training, evaluation, and deployment of the detection pipeline.
  • Generated visualizations of model predictions and detection accuracy for reporting and presentation.
