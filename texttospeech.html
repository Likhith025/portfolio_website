Text-to-Speech Converter

Jan 2024 – Feb 2024

Developed a basic Text-to-Speech web application using HTML, CSS, and JavaScript. The application leverages the browser's built-in speech synthesis API to convert user-input text into audible speech. This project demonstrates understanding of DOM manipulation, event handling, and integration of web APIs.

  • Implemented a clean UI with a textarea for text input and buttons to play, pause, and stop speech.
  • Used the Web Speech API for converting text to speech with selectable voices supported by the browser.
  • Added functionality to adjust speech rate and pitch for a customizable experience.
  • Ensured cross-browser compatibility for major browsers supporting the Web Speech API.
  • Created a lightweight, dependency-free application suitable for learning and demonstration purposes.
