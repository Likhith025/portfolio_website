Timetable Generator

Feb 2025 – June 2025

Designed and developed a web-based Timetable Generator that uses a genetic algorithm to automatically generate conflict-free schedules based on user-defined constraints such as subjects, faculty availability, and classroom allocations.

  • Implemented a genetic algorithm to optimize scheduling and ensure minimal conflicts.
  • Built a React.js frontend for timetable creation, editing, and real-time visualization.
  • Developed a secure Node.js + Express backend with MongoDB for storing timetable data and user configurations.
  • Integrated JWT authentication and Google OAuth for secure user login and third-party authentication.
  • Deployed the backend on Render and the frontend on Vercel with environment-based configurations.
  • Collaborated via Git & GitHub for version control, CI/CD, and team project management.

Tech Stack Used:

React.js
Node.js
Express.js
MongoDB
OAuth
Git
Vercel
